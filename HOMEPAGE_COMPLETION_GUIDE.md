## 0. 반드시 지켜야 할 기본 원칙

1. 모든 대화와 산출물은 한국어로 작성한다.
2. 사용자는 박진석 이사님이며, 호칭은 "박이사님"으로 한다.
3. 썬레이텍의 핵심 제품은 원적외선 방사 천장형 복사난방패널이다.
4. 조달, 인증, 시험성적, 에너지 절감 수치는 반드시 원천자료 또는 현재 코드/DB의 검증된 자료에 근거한다.
5. 홈페이지 내 회사 대표 도메인 표시는 현재 운영 중인 `www.sunraytec.co.kr`을 기준으로 한다.
6. `www.sunraytec.kr`은 향후 대표 도메인 전환 후보로만 기록하고, 현재 콘텐츠/SEO 기준 도메인으로 쓰지 않는다.
7. `sunraytec.net`은 현재 이메일 도메인으로 보유·운영 중이며, 대표 홈페이지 도메인과 혼동하지 않는다.
8. 우수제품 콘텐츠는 홈페이지 완성/출시 시점에 제3자단가계약 완료 및 나라장터 종합쇼핑몰 등재가 완료된다는 전제로 작성한다.
9. 다만 실제 공개 직전에는 계약 완료 여부, 종합쇼핑몰 등재 여부, 제품 식별번호를 최종 검수한다.
10. MAS 제품은 2026-05-04 나라장터 종합쇼핑몰에서 `썬레이텍`으로 검색 확인한 9개 식별번호를 현재 기준값으로 사용한다.
11. 모바일 화면에서 텍스트, 태그, 표, 다이어그램이 잘리는 상태는 출시 불가로 본다.
12. `overflow-x: hidden`으로 문제를 숨기지 말고, 실제 레이아웃 원인을 수정한다.
13. 기존 사용자/다른 AI의 변경사항은 되돌리지 않는다.
14. 코드 변경 후에는 가능한 한 `npm run lint`, `npm run build`, 로컬 브라우저 확인을 수행한다.
15. 홈페이지 공개 문구에는 `수의계약 가능`, `입찰 없이 구매 가능`처럼 법적 오해가 생길 수 있는 표현을 쓰지 않는다.

- 그러나 홈페이지 완성 시점에는 계약 완료 및 나라장터 종합쇼핑몰 등재가 완료된다는 전제로 우수제품 콘텐츠를 작성한다.
- 따라서 고객-facing 콘텐츠는 "계약 완료 후 운영 상태"를 기준으로 설계한다.
- 단, 실제 배포 직전에는 계약 완료 여부와 종합쇼핑몰 등재 상태를 반드시 최종 확인한다.

표현 원칙:

- 공개 완료 버전 기준 표현: "조달청 우수제품", "제3자단가계약 제품", "나라장터 종합쇼핑몰 등록 제품", "공공기관 구매 상담 가능"
- 내부 개발/검수 메모: "현재 신청서류 작성 중", "출시 전 계약/등재 최종 확인 필요"
- MAS 표현: 2026-05-04 나라장터 종합쇼핑몰 검색 기준 9개 식별번호를 현재 기준값으로 사용하고, 공개 직전 재검수
- 구매 방식 표현: `수의계약 가능` 대신 `구매 금액과 수요기관 기준에 따라 2단계경쟁 또는 바로 납품요구 적용 가능`으로 안내

완료 기준:

- 고객-facing 페이지는 출시 시점 계약 완료/등재 상태 기준으로 자연스럽게 작성되어 있다.
- 내부 TODO 또는 지침에는 공개 전 최종 검수 항목이 남아 있다.
- 제품별 조달 상태, 식별번호, 등록 여부가 데이터로 구분되어 있다.
- `src` 전체에서 과장·오해 가능 문구 검색 후 검토 완료.

완료 기준:

### 확인 완료

5. 우수제품 제3자단가계약은 현재 신청서류 작성 중이나, 홈페이지 완성 시점에는 계약 완료 및 종합쇼핑몰 등재를 전제로 콘텐츠를 작성한다.
