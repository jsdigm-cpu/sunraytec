## 3. 발견된 문제점 및 개선 우선순위

### 🔴 P0 — 긴급 (박이사님 직접 지적 사항)

#### 문제 1: 시공사례 페이지 — Hero 아래 너무 어둡고 사진까지 스크롤이 너무 많음

**현상**:
- `/cases` 페이지에서 SubHero 아래로 진한 네이비(#0D1B2E) 배경 섹션이 연속 2개 나옴
  1. KPI 수치 바 (어두운 배경)
  2. 대표 실증 사례 Spotlight 2건 (어두운 배경)
- 이 두 섹션을 통과해야 비로소 카테고리 인포그래픽과 사례 사진 그리드가 나옴
- 결과적으로 실제 시공 사진을 보려면 스크롤을 화면 4~5개 분량 해야 함

**권장 처방**:
- KPI 수치를 SubHero 내부 또는 서브메뉴 바 형태로 통합 (별도 dark 섹션 제거)
- Spotlight 섹션을 갤러리 그리드 하단으로 이동 (또는 사이드 패널로 전환)
- 페이지 진입 시 바로 카테고리 탭 + 사례 사진 그리드가 노출되도록 순서 재배치

#### 문제 2: 전국 시공 지도 — 실제 지도 없음

**현상**:
- `/cases-map` 메뉴는 있지만 실제 지도 API(카카오맵, 네이버맵 등)가 없음
- 텍스트 목록으로 권역별 현장을 나열하는 방식만 구현됨
- "전국 시공 지도"라는 메뉴명과 기대치 불일치

**권장 처방**:
- 카카오맵 JavaScript API 연동 (무료 할당량 충분)
- 각 시공 현장 위도/경도 데이터를 Supabase case_studies 테이블에 추가
- 지도 위에 카테고리별 마커 + 클릭 시 팝업으로 현장 정보 표시

#### 문제 3: 모바일 서브메뉴 — 풀다운(아코디언) 형식 미적용

**현상**:
- 현재 모바일 메뉴는 햄버거 버튼 클릭 → 전체 메뉴 펼침 (모든 서브메뉴도 함께 펼쳐짐)
- 6개 메인 카테고리 × 6개 서브메뉴 = 36개 항목이 한 번에 나열됨
- 원하는 메뉴 항목 찾기 불편하고 스크롤 부담 과다

**권장 처방**:
- 메인 카테고리 클릭 → 서브메뉴 토글(아코디언) 방식으로 변경
- ChevronDown 아이콘 회전 애니메이션으로 열림/닫힘 상태 표시
- 현재 활성 카테고리는 기본 열린 상태로 표시

#### 문제 4: 기술·솔루션 페이지 — "현장에서 먼저 해결해야 할 문제" 섹션 간헐적 미로딩

**현상**:
- `/technology/zero`, `/solutions/*` 페이지에서 Problem Map 섹션이 가끔 표시 안됨
- 새로고침 시 정상 표시됨
- `motion/react`의 `whileInView` + `viewport={{ once: true, amount: 0.15 }}` 조합의 타이밍 이슈로 추정
  - `staggerContainer` → `staggerItem` 중첩 애니메이션에서 첫 진입 시 뷰포트 감지 누락

**권장 처방**:
```tsx
// 현재 코드 (문제 있음)
viewport={{ once: true, amount: 0.15 }}

// 권장 수정
viewport={{ once: true, amount: 0.05, margin: "0px 0px -50px 0px" }}
```
- `amount` 임계값을 0.15 → 0.05로 낮춰 더 빨리 트리거
- `margin` 음수값으로 뷰포트보다 50px 먼저 트리거

#### 문제 5: 기술·솔루션 컨텐츠 — 형식적이고 성의없어 보임

**현상**:
- 5개 솔루션 페이지(`zero`, `public-edu`, `industrial-logistics`, `defense-special`, `iot-control`)가 동일한 PAGES 객체 구조에 다른 값만 채워 넣은 방식
- Problem Map → Spotlight → Method → Applications → Checklist → CTA 구조가 5개 페이지 전부 동일
- 각 섹션의 내용이 실제 현장 경험에서 나온 느낌보다 템플릿을 채워 넣은 느낌

**권장 처방**:
- 각 솔루션 페이지에 해당 분야만의 독특한 시각 요소 추가
  - 공공·교육: 조달 흐름도(나라장터 → 수의계약 → 납품)
  - 산업·물류: 에너지 절감 계산기 인터랙션 (면적·층고 입력 → 예상 절감액)
  - 국방·특수: 방폭 인증 마크 + 혁신제품 지정서 이미지
- "현장에서 먼저 해결해야 할 문제" 각 항목에 실제 현장 사진 추가 (적용 전/후)
